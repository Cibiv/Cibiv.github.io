Simulating and Estimating Horizontal Gene Transfer

Introduction:

We provide a software package to measure horizontal gene transfer.

Reference:

The method is described in detail:
  • Simone Linz, Achim Radtke, Arndt von Haeseler: A Likelihood Framework to Measure Horizontal Gene Transfer, Molecular Biology and Evolution, 24:1312-1319, 2007.
