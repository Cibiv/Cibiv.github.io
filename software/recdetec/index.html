RecDetec - Detecting Recombination and Phylogenetic Information Along Alignments

Introduction:

RecDetec is a software to investigate recombination scenarios based on the bootscan principle along alignments of (viral) genomes. It also allowes to assess the content of phylogenetic information along alignments.

Further details on how to install, run, and use this program can be found in the RecDetec manual (PDF).
